# 回流 和 重绘

  • 回流-reflow
    当render树中的一部分或者全部因为大小边距等问题发生改变 而需要重建的过程叫做回流。

  • 重绘-repaint
    当元素的一部分属性发生变化，如外观背景色不会引起布局变化 而需要重新渲染的过程叫做重绘

# display:none 和 visibility:hidden

  • display:none 元素彻底消失，不在文档流中占位，浏览器也不会解析该元素
    会引起 回流 和 重绘

  • visibility:hidden
    视觉上消失了，相当于透明度opacity:0，在文档流中占位，浏览器也会解析
    会引起 重绘

TIP

回流必将引起重绘，而重绘不一定会引起回流
